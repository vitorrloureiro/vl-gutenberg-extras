This plugin adds extra functionality to core Gutenberg blocks.

== Description ==
Sometimes you don\'t need whole new blocks to create the designs you want in the block editor.

This plugin adds the following set of functionalities to the core Gutenberg blocks:

= All blocks =
Hide on mobile
Hide on desktop

= Image =
Image fit cover
Image fit contain

= Button =
Animation on hover: shake



== Installation ==
Simply install and activate the plugin.
